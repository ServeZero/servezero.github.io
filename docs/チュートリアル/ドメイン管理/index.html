ドメイン管理

概要​

ServeZeroではWebアプリケーションをドメインまたはサブドメインのサイト単位で管理します。

準備​

あらかじめ構築するサイトのドメイン名をDNSを管理しているサービスの管理画面から追加します。
DNSの設定にドメイン名とServeZeroのサーバのIPをAレコードで追加します。

ここではwww.example.comドメインを追加する例を示します。

注意

DNSの設定がインターネットに反映するのにはしばらく時間がかかります。

ドメインの追加​

ドメイン一覧ページを開きます。現在はサインイン直後のトップページがドメイン一覧ページです。
右上の新規ボタンから新規サイトの情報を入力するダイアログを開きます。

ドメイン名にwww.example.com入力し、Webアプリケーションを選択してドメインを追加ボタンを押します。

ドメイン一覧​

しばらく待つと一覧にドメインが追加されます。
一覧の操作項目は、左から詳細ボタン、削除ボタンが並びます。

ドメイン情報​

詳細ボタンからドメイン情報ページを開きます。

Webアプリケーションのインストール​

新規作成したサイトのWebアプリケーションのインストール作業を行います。
WebブラウザでドメインのURLhttp://www.example.comに直接アクセスしてインストーラを開きます。

ドメイン情報ページを参照しながら必要な情報を入力し、インストール作業を完了します。
