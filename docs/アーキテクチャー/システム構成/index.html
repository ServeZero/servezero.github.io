システム構成

概要​

ServeZeroシステムは、ドメインのホスティング機能とホスティング管理機能の２つの機能で構成されています。
ホスティング機能はDockerコンテナ上で動作するサービスで、ホスティング管理機能はサーバのネイティブアプリケーションとして動作するサービスです。
